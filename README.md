# TW-Kai-webfont

再封裝[全字庫正楷體](https://cns11643.gov.tw/)，使其得適應於網路傳輸環境，可作為網頁字型使用。

## 嵌入專案
若要在網頁上使用全字庫正楷體，請將此段代碼複製到 HTML 的 <head> 標籤。 
```html
<link href="https://cdn.jsdelivr.net/gh/Aerotinge/TW-Kai-webfont/TW-Kai_font.css" rel="stylesheet">
```
